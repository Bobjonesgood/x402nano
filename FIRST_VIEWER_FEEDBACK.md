# First Viewer Feedback

Use this when showing LeadNestAI to someone for the first time.

The goal is not to convince them immediately. The goal is to learn where the explanation is clear and where it still creates friction.

## 1. Before Showing It

Open the live demo:

```txt
https://x402nano.onrender.com
```

Keep the explanation short:

```txt
This is LeadNestAI. It is a lead intelligence demo where the API can ask for payment and unlock a premium lead pack after payment proof.
```

Then stop talking and let them look.

## 2. Watch For Hesitation

Pay attention to where they pause:

- top headline
- "How LeadNestAI Works"
- unlock button
- sandbox disclosure
- receipt panel
- lead intelligence cards
- protocol trace
- FAQ section

If they hesitate, write down the exact section where it happened.

## 3. Ask These Questions

Start with:

```txt
What do you think this does?
```

Then ask:

```txt
What do you think the payment unlocks?
```

Then ask:

```txt
What do you think "machine-payable" means here?
```

Then ask:

```txt
Was it clear that this is sandbox settlement and no real money moves yet?
```

Then ask:

```txt
What would make the lead pack more valuable?
```

## 4. Feedback Notes Template

Copy this for each conversation:

```txt
Viewer:
Date:
Audience type:

Could they explain the demo in their own words?

Did they understand what unlocks?

Did they understand sandbox settlement?

What confused them first?

What part felt strongest?

What business use case did they mention?

What wording should change?

Next improvement:
```

After the conversation, copy the useful notes into:

```txt
FEEDBACK_ITERATION_LOG.md
```

## 5. Top Confusion Points To Track

Track these patterns:

- They think this is already real revenue.
- They think x402 is the product instead of the infrastructure.
- They do not understand what a lead intelligence pack contains.
- They miss the receipt panel.
- They do not understand why the API returns `402 Payment Required`.
- They understand the tech but not the business use.
- They understand the business use but not the sandbox boundary.

## 6. Success Signal

The demo is working if a first-time viewer can say something close to:

```txt
This is a lead intelligence API that can charge for a premium lead pack and unlock it automatically after payment.
```

That is the comprehension target.
